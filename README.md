## 구현 기능
- **원카드 시스템**
  - 기본적인 원카드 시스템을 구축
    - 카드를 냈을때 로직
    - 승리/패배 로직
    - 카드를 N장 가져오는 로직
- **네트워크 동기화**
  - 플레이어가 카드를 냈을때 해당 정보를 동기화하여 보여주는 시스템
